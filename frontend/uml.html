--- 
		title: Sequence - UML-diagram 
		---

		flowchart TB
			subgraph Start Game
				A[User Logs In] --> B[Navigate to Game Lobby]
				B --> C[Select Team]
				C --> D{Enough Users?}
				D -- Yes --> F[Click Start Game]
				D -- No --> E[Wait for More Users]
				E --> D
	  	end
	  
		  subgraph Main Loop
				F --> G[Player Plays Card]
				G --> H[Player Places Chip]
				H --> I[Check for Sequence]
				I --> J{Game Over?}
				J -- No --> K[Player Draws New Card]
				K -- Next Player --> G
		  end
	  
		  subgraph End Game
				J -- Yes --> L[Display Winner]
				L --> M[Start New Game]
		  end
